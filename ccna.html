Interactive Guide to the CCNA 200-301 Exam

An upgraded, comprehensive overview of the skills required for Cisco's premier networking certification.

Start Exploring

1.0 Network Fundamentals (20%)

This domain covers the foundational knowledge of network components, architectures, and the core principles of IP addressing and connectivity.

1.1 Network Components

  • Explain the role of Routers & L2/L3 Switches
  • Describe Next-gen firewalls and IPS
  • Understand Access points, Controllers, Endpoints, and Servers

1.2 Topologies

  • Compare Two-Tier, Three-Tier, and Spine-Leaf
  • Describe WAN and SOHO architectures
  • Differentiate On-premises vs. Cloud

1.3-1.5 Physical & Transport

  • Compare fiber, copper, and connection types
  • Identify interface issues (collisions, errors, duplex)
  • Contrast the roles and functions of TCP and UDP

1.6-1.7 IPv4 Addressing

  • Configure and verify IPv4 addressing and subnetting
  • Describe private IPv4 addressing schemes (RFC1918)

1.8-1.10 IPv6 Addressing

  • Configure and verify IPv6 addressing and prefix
  • Describe IPv6 address types (Unicast, Anycast, Multicast)
  • Verify IP parameters on client operating systems

1.11-1.13 Core Concepts

  • Explain wireless principles (Channels, SSID, RF)
  • Describe virtualization fundamentals (VMs, Containers, VRFs)
  • Understand switching concepts (MAC learning, flooding)

2.0 Network Access (20%)

Focuses on configuring and verifying VLANs, interswitch connectivity, Layer 2 protocols, and WLAN components.

2.1 VLANs & InterVLAN Routing

  • Configure VLANs (normal range) across switches
  • Set up access ports for data and voice
  • Implement InterVLAN connectivity

2.2 Interswitch Connectivity

  • Configure 802.1Q trunk ports
  • Understand the role of the Native VLAN

2.3-2.4 L2 Protocols

  • Configure and verify CDP and LLDP
  • Configure L2/L3 EtherChannel using LACP

2.5 Rapid PVST+

  • Interpret root bridge, port roles, and states
  • Understand PortFast, BPDU Guard, and other features

2.6-2.7 Wireless Architecture

  • Describe Cisco Wireless Architectures and AP modes
  • Explain physical connections for WLAN components (AP, WLC)

2.8-2.9 Device & WLAN Access

  • Describe device management access methods
  • Interpret wireless LAN GUI for client connectivity

3.0 IP Connectivity (25%)

The largest domain, covering routing tables, how routers make forwarding decisions, static routing, and single-area OSPFv2.

3.1 Routing Table Components

  • Interpret protocol code, prefix, and mask
  • Identify next hop, administrative distance, and metric
  • Understand the gateway of last resort

3.2 Router Forwarding Decision

  • Explain how a router uses the longest prefix match
  • Describe the roles of AD and metric in path selection

3.3 Static Routing

  • Configure and verify IPv4/IPv6 static routes
  • Implement default, network, host, and floating static routes

3.4 Single-Area OSPFv2

  • Configure neighbor adjacencies
  • Set up OSPF on point-to-point and broadcast networks
  • Understand DR/BDR selection and Router ID

3.5 First Hop Redundancy

  • Describe the purpose and concepts of FHRPs like HSRP and VRRP

4.0 IP Services (10%)

Covers essential network services including NAT, NTP, DHCP, DNS, SNMP, QoS, and secure remote access.

4.1-4.3 Core Services

  • Configure and verify inside source NAT (static and pools)
  • Configure NTP in client and server mode
  • Explain the roles of DHCP and DNS

4.4-4.6 Management & Monitoring

  • Explain the function of SNMP
  • Describe syslog features (facilities, levels)
  • Configure a DHCP client and relay agent

4.7-4.9 QoS & File Transfer

  • Explain QoS concepts (classification, marking, queuing)
  • Configure network devices for remote access using SSH
  • Describe the use of TFTP/FTP in the network

5.0 Security Fundamentals (15%)

Introduces key security concepts, access control, VPNs, Layer 2 security features, and wireless security protocols.

5.1-5.4 Device & Password Security

  • Define threats, vulnerabilities, and exploits
  • Configure device access control with local passwords
  • Describe password policy elements (MFA, biometrics)

5.5-5.6 VPNs & ACLs

  • Describe IPsec remote access and site-to-site VPNs
  • Configure and verify standard and extended ACLs

5.7-5.10 L2 & Wireless Security

  • Configure DHCP Snooping, Dynamic ARP Inspection, Port Security
  • Compare AAA concepts
  • Describe WPA, WPA2, and WPA3 protocols
  • Configure a WLAN using WPA2 PSK in the GUI

6.0 Automation and Programmability (10%)

Explores the shift towards modern networking with controller-based architectures, APIs, and configuration management tools.

6.1-6.3 Controller-Based Networking

  • Explain how automation impacts network management
  • Compare traditional vs. controller-based networks
  • Describe SDA architecture (overlay, underlay, fabric)

6.4-6.5 APIs & AI/ML

  • Explain the role of AI and ML in network operations
  • Describe REST-based APIs (CRUD, HTTP verbs, data encoding)
  • Understand Northbound and Southbound APIs

6.6-6.7 Config Management & Data

  • Recognize capabilities of Ansible and Terraform
  • Recognize components of JSON-encoded data
